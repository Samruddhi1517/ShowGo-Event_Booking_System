# ShowGo 🎟️

### Event Management & Booking Website

ShowGo is a modern and responsive event booking web application where users can explore events, view details, select seats, make payments, and download tickets.
It also includes an Admin Panel to add, update, and remove events dynamically.

---

# ✨ Features

## 👤 User Features

* Search & filter events
* Event details page
* Interactive seat booking UI
* VIP / Gold / Silver pricing
* Payment page
* Ticket confirmation page
* Download ticket
* Responsive modern UI

---

## 🔐 Admin Features

* Admin login
* Add events dynamically
* Edit events
* Delete events
* Drag & drop image upload
* LocalStorage-based data management

---

# 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* LocalStorage
